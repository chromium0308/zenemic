# Zenemic

### Plan any event in a sentence.

Zenemic is an AI-powered event planner that turns a plain-language description into a fully
organized event — calendar invites, a fair bill split, directions, and a smart timeline — without
the back-and-forth.

## What is Zenemic?

Organizing a get-together usually means juggling a group chat, a calendar app, a maps app, and a
"who owes what" spreadsheet. Zenemic collapses all of that into a single step: **describe the plan
the way you'd say it out loud**, and let the app handle the logistics.

> *"Friday 8pm dinner at Bottega with Sam, Alex and Priya — split the £180 bill."*

From one sentence, Zenemic understands the **what, when, where, who, and how much** — then sets up
everything needed to make it happen and keep everyone on the same page.

## Features

- 🧠 **Natural-language planning** — describe an event in your own words; Zenemic extracts the
  details and fills in the rest.
- 📅 **Calendar, sorted** — generates the event and invites so it's on everyone's calendar.
- 💸 **Effortless bill splitting** — splits costs across guests and sends payment requests, with
  receipt scanning to itemize the bill.
- 🗺️ **Getting there** — directions and public-transit links to the venue, built in.
- 🧭 **Smart timelines** — an at-a-glance plan for the day, from setup to wrap-up.
- 💬 **Ask Zenemic** — a per-event AI assistant for quick questions and nudges ("who hasn't paid?").
- 📸 **Shared albums** — a place for everyone's photos from the event.

## How it works

1. **Describe** — type (or speak) what you're planning.
2. **Review** — Zenemic drafts the event, split, and route for you to tweak.
3. **Confirm** — send the invites and payment requests in a tap.

## The Zenemic Keyboard

Zenemic also reimagines event planning as something you can do **from anywhere**. A custom keyboard
lets you spin up a full event right inside whatever app you're already in — describe the plan, and
the same intelligence does the rest, then hands you a result you can review before confirming.

## Platforms

Zenemic is cross-platform, with a **public beta arriving soon on iOS and Android**.

## Built with

Zenemic is powered by **Anthropic's Claude** models, with a React Native (Expo) app and a
TypeScript backend. It integrates with services people already use — Google Calendar, Google Maps,
Stripe, and Transport for London — so plans flow into the tools you rely on.
